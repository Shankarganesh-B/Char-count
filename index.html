Char counter

Coded By Shankarganesh

Key features:

  • The char limit is not pre-set. Users have the option to set it. This ensures that, this could be repurposed when required.
  • Everytime when user sets max char limit, the box resets. This helps when users give input, without setting max value.
  • The char limit cannot be zero or less. This ensures that the max char limit remains positive
